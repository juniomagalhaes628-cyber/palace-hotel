# 🏨 Palace Hotel & Spa · Termas de São Vicente

Um website profissional e animado para o **Palace Hotel & Spa Termas de São Vicente**, localizado em Pinheiro, Portugal.

## ✨ Características

- 🎨 **Design Elegante**: Tema escuro com acentos quentes inspirado em Grand Emily Hotel
- ✅ **Animações Suaves**: 
  - Zoom de herói com contador animado (0→100%)
  - Revelaçao de palavras sequenciais
  - Scroll reveal com efeito stagger
  - Hover effects magnéticos no menu
  - Parallax scrolling

- 🎯 **Funcionalidades Completas**:
  - Navegação fixa com efeito scroll
  - Booking inline com seletores de datas
  - Galeria de quartos por "Alas" (Charme e Histórica)
  - Informações de serviços spa
  - Avaliações com rating 4★ (978 reviews)
  - Mapa Google Maps integrado
  - Formulário de contacto

- 📱 **Responsivo**: Design mobile-first

## 🚀 Como Utilizar

```bash

git clone https://github.com/juniomagalhaes628-cyber/palace-hotel.git

# Abra palace-hotel.html num navegador
# ou use um servidor local na porta 3090
```

### Funcionalidades do Site

1. **Navegação**: Clique em SOBRE NÓS, HOTEL, TERMAS & SPA, etc.
2. **Booking**: Preencha as datas e número de hóspedes
3. **Galeria de Quartos**: Hover para ver zoom e detalhes
4. **Reviews**: Veja avaliações reais do Google (978 reviews, 4★)
5. **Contacto**: Encontre localização, telefone e email

## 🎬 Animações Implementadas

- **Page Loader**: Contador de 0 a 100% no carregamento
- **Hero Title**: Cada palavra entra sequencialmente
- **Scroll Reveals**: Elementos aparecem com stagger delays
- **Parallax**: Imagens movem-se conforme scroll
- **Hover Effects**: Cards dos quartos com zoom na imagem
- **Shimmer**: Efeito brilho nos botões
